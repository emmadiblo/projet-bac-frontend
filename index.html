      Aide et Documentation

      Cette application permet de gérer automatiquement les emplois du temps et les salles de cours dans une faculté. Elle offre une interface intuitive pour planifier les cours, gérer les ressources et résoudre les conflits d'horaires.

      Comment utiliser cette application :

      • Tableau de bord : Vue d'ensemble des emplois du temps et des statistiques
      • Gestion des salles : Ajouter, modifier ou supprimer des salles
      • Gestion des enseignants : Gérer les informations des professeurs
      • Gestion des cours : Configurer les cours et leurs attributs
      • Génération des emplois du temps : Créer automatiquement des emplois du temps

      Pour plus d'informations, consultez la documentation complète.
